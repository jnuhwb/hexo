H.264基本概念

I帧： 关键帧，帧内压缩

P帧：向前参考帧，压缩时只参考前一个帧，帧间压缩

B帧：双向参考帧，压缩时参考前后一帧，帧间压缩

GOP：两个I帧之间的帧分组

SPS：Sequence Parameter Set，序列参数集，帧数，参考帧数目，解码图像尺寸，帧场编码模式选择标识等

PPS：Picture Parameter Set，图像参数集，熵编码模式选择标识，片组数目，初始量化参数，去方块滤波系数调整标识等

花频/卡顿：GOP分组中P帧或者I帧丢失，就丢弃本GOP内所有帧，直到下个I帧

整数离散余弦变换（DCT），将空间上的相关性变为频域上的无关的数据然后进行量化，（理解傅立叶变换，小波变换有助于理解这个）

码流基本概念

SODB，String Of Data Bits 原始数据比特流，长度不一定是8的整数倍，是由VCL产生的

RBSP，Raw Byte Sequence Payload => SODB + trailing bits，SODB后补1，不按字节对齐再补0

EBSP，Encapsulate Byte Sequence Payload，0x00 00 00 01或0x00 00 01，压缩后数据出现两个连续0x00则插入0x03

NALU，NAL Header(1B) + EBSP

  • You own any object you create. You create an object using a method whose name begins with “alloc”, “new”, “copy”, or “mutableCopy”

iOS自动构建命令小结

发表于 2015-07-14 | 分类于 iOS

简单讲就是xcodebuild编译，codesign签名，xcrun打包。

一、编译

xcodebuild -workspace xx.xcworkspace -scheme xx -sdk iphoneos -configuration Release SYMROOT=XX DEPLOYMENT_PROCESSING=YES PROVISIONING_PROFILE=XX

如果没有用工作空间

xcodebuild -target xx -sdk iphoneos -configuration Release SYMROOT=XX DEPLOYMENT_PROCESSING=YES PROVISIONING_PROFILE=XX

SYMROOT

指定输出目录

DEPLOYMENT_POSTPROCESSING

DEPLOYMENT_POSTPROCESSING 部署后处理，就是Xcode build settings中的Deployment Postprocessing，一般为NO，此参数受ACTION的影响，当$ACTION=install时默认为YES

如果不设置DEPLOYMENT_POSTPROCESSING=YES，则编译后的APP会比手动打包大很多，猜想手动打包Archive中包含install？

PROVISIONING_PROFILE

就是APP包里的embedded.mobileprovision文件，不指定就会使用工程配置。使用Provisioning Profiles的UUID。

获取UUID的方式一

Build Settings => Provisioning Profile => Release => 点击右栏，弹出选择框 => 选择想要查看UUID的文件 => 再次点击右栏，弹出选择框 => 点击Other… 就会弹出已选择文件的UUID

获取UUID方式二

Xcode => Preferences => Accounts => 选择你的开发帐号 => View Details => Provisioning Profiles => 选择想要查看UUID的文件，右键Show in Finder，该文件名称就是它的UUID

二、签名

codesign -f -s identity –entitlements entitlements_path app_path
或者
codesign –force –sign identity –entitlements entitlements_path app_path

identity

identity可以用证书名称或者SHA1 fingerprint，都可以从Keychain中相应证书的简介中找到。

entitlements

一开始没有指授权文件，打出来的IPA包在iOS8以下可以安装，在iOS8以上就无法安装了，从这个贴子得到启发https://devforums.apple.com/message/1096945#1096945

找到苹果文档说明

Set entitlement values in order to enable iCloud, push notifications, and App Sandbox

一个entitlements例子，把TeamIdentifier换成你的开发证书ID

三、打包

xcrun -sdk iphoneos PackageApplication -v app_path -o output_pa_path
